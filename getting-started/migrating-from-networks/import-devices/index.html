Import End Devices in The Things Stack

Command-line interface only

The instructions below use the command-line interface (CLI).

Learn how to install the CLI

To import your devices, you need an Application in The Things Stack. This can be done by following instructions for Creating an Application.

To import devices, use the devices.json file you created in the previous step with ttn-lw-cli:

$ ttn-lw-cli end-devices create --application-id "application-id" < devices.json

This will import your devices on The Things Stack. In case any device fails, you will see a relevant error message at the end of the output.

If the import is successful, your devices are added to the list of end devices in your application.

If the import was successful, your devices are added to the list of end-devices in your application.
